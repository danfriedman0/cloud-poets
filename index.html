rnn-js

about

This is a demo of an LSTM recurrent neural network that's been trained to write poetry like Alexander Pope. I used Andrej Karpathy's char-rnn code to train the model in Torch, I used Davide Testuggine's fork of char-rnn to convert my trained model to JSON and then I used Karpathy's recurrentjs library to sample from the model in JavaScript.

char-rnn is a pretty cool project. The model works by predicting one character at a time. For more you should look at Karpathy's repository and also his article, "The Unreasonable Effectiveness of Recurrent Neural Networks," which has more examples. My purpose here was to wrap some of these other libraries to make it even simpler to train a model using Karpathy's Torch code on a GPU and then use JavaScript to sample from it in the browser. I have more RNN poets in the works.
